Ros系统安装教程

作者：寂

安装说明

安装环境：Ubuntu16.04LTS 安装的Ros版本：Kinetic
Ros版本 对应Ubuntu版本
Melodic Ubuntu18.04
Kinetic Ubuntu16.04
Indigo Ubuntu14.04

安装准备

安装好Ubuntu16.04LTS，可以是在虚拟机上，也可以是笔记本或者台式机直接安装。
要求有一定的Linux操作基础，知道一些基础命令。

安装开始

一、更换软件源

这一步非常重要。软件源原本默认是Ubuntu官方的软件源。现在要安装Ros系统，我们还需要添加Ros系统的软件源。
可以用快捷键Ctlr+Alt+T打开终端，或者在桌面鼠标右键打开终端。
需要修改软件源的源文件。文件目录为：/etc/apt/source.list。
可以使用Tab键补齐命令，建议在改源之前先备份文件。

方式一：官方源

在终端直接输入命令：
$ sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'

方式二：国内中科大源

这里建议只用国内的软件源，下载会更快。
$ sudo sh -c '. /etc/lsb-release && echo "deb http://mirrors.ustc.edu.cn/ros/ubuntu/ $DISTRIB_CODENAME main" > /etc/apt/sources.list.d/ros-latest.list'
完了之后输入密钥：
$ sudo apt-key adv --keyserver hkp://pool.sks-keyservers.net --recv-key 0xB01FA116
如果上面那个不行，再试试这个：
$ sudo apt-key adv --keyserver hkp://ha.pool.sks-keyservers.net:80 --recv-key 421C365BD9FF1F717815A3895523BAEEB01FA116
成功了之后就更新一下软件包：
$ sudo apt-get install update

二、安装Ros软件包

安装的形式很简单：sudo apt-get install ros-版本-desktop-full
上面这个是完整安装，还有一个不完整安装就是把后面的full去掉。
所以我们安装Kinetic版本的命令就是：
$ sudo apt-get install ros-Kinetic-desktop-full
安装一些依赖工具包：
$ sudo apt-get install python-rosinstall python-rosinstall-generator python-wstool build-essential
初始化
$ sudo rosdep init
$ rosdep update

配置环境变量

$ echo "source /opt/ros/kinetic/setup.bash" >> ~/.bashrc
$ source ~/.bashrc

测试

在终端输入：roscore -h
如果没有报错则安装完成 输入下面这个命令可以启动Ros系统。
$ roscore
打开一个新的终端，输入下面的命令，还可以出现一个随机的乌龟：
$ rosrun turtlesim turtlesim_node
